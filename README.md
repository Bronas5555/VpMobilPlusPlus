# VpMobil++
A alternative client for the VpMobil Application written with [Avalonia UI](https://avaloniaui.net/) in C#.

## Supported Plattforms
- Windows 64
- Linux 64
- Android

iOS and Web-Assembly projects are available, but aren't tested.


## Features
- List of classes with pin function
- Viewing of plans
- Portrait / landscape support
- Filtering of subjects
- Custom colors for subjects

## Security Warning
Every Version before Alpha v0.3 stores Login Information in plain text on your device.
